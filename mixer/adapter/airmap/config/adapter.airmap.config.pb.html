The airmap adapter is designed to bridge over to an in-cluster adapter instance via ReST.

Params

Configuration format for the AirMap adapter.

Field Type Description
endpoint string

The endpoint implementing access.Controller
amqpUsername string

The username for the amqp producer
amqpPassword string

The password for the amqp producer
amqpHost string

The host for the amqp producer
amqpPort int32

The port for the amqp producer
amqpQueuename string

The queue name to push logs to on the amqp endpoint
